Scrolls the screen when you click a menu link

See http://quasardesenvolvimento.com.br/wordpress-scrollspy-menu/ for full documentation.

== Installation ==

1. Upload the entire 'scrollspy' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
